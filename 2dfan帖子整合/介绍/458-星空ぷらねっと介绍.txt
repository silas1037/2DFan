
                
　　记得小时候一直喜欢看着天上的星星，在看多的时候就会去想，究竟有没有月亮上的嫦娥，或者银河两岸的牛郎织女呢（笑）？等我大一点的时候就只想去买一个天文望远镜，结果访遍家乡的大街小巷也找不到，自己后来用一个放大镜的镜头和外婆的老花镜片加上一捆硬纸板也做成了一个简陋的天文望远镜。当第一眼看见月亮上的环形山的时候，当亲眼看见天狼星那个16等的白矮伴星的时候，兴奋得好几个晚上无法睡着。现在，当我走过商店边瞥见里面堂而陈列的各式价格形状的天文望远镜的时候，却再也提不起兴趣去买了。晚上再次抬头看看星空，忽然想笑一笑，其实，有的东西永远留在内心中，也许更美好吧！
 ; ; ; 当我拿到这个游戏的时候，忽然仿佛很有感触一般，总觉得年少的我也在游戏中活了起来，宛如片头的图片，在夜晚的星空下，数着愿望的男孩和女孩。可惜年少时曾经一起徘徊在夜空下的那个女孩已经不知道去向了——也许，她现在和另外一个男孩正在数星星吧…… ; ; ; 故事中的男孩也许更喜爱星空一点，在他好小的时候，因为受到母亲的影响，一直憧憬着星空里的神秘，也一直做着当上一名宇宙飞行员的梦。……还是在那个时候，身边围绕过各种各样的女孩，有那位一口关西腔老是吵吵闹闹的家伙，有那位总是不喜欢笑并且还老揪别人小辫子的学习委员，那个总是温柔地笑的大姐姐，那个总是沉默寡言的小妹妹，对了，还有一个已经忘了笑容的她……每天和她们一起，数着星星，吹嘘着自己也许真的会有一天实现的梦想，真的很美。
 ; ; ; ……………… ; ; ; 真的一直都很美，至少在那一天到来之前，当空中那团火球爆发出来之前，当心中只能回荡起那声震耳欲聋的爆炸声的时候，亲人和少年的梦想……都在瞬间破灭了。 ; ; ; 失去梦想的少年，再也没有勇气仰望着星空，他远远地离开了曾经埋葬了少年时候梦想的街道，逃避着无法逃避的自己。 ; ; ; 于是，过了七年…… ; ; ; 少年再次回到了记忆中的伤心的那片空地，在这里，少男少女的面容再次重叠…… ; ; ; “如果下次碰面的时候，我会笑给你看……”在七年前，那个从来没有笑过的女孩在分手时候说到。 ; ; ; 少年想笑着，却只能留下泪水…… ; ; ; “你还会仰望星空吗？” ; ; ; “你还会憧憬着年少时痴狂的梦想吗？……” ; ; ; “如果，你真的想哭的话，请靠到我的怀中吧…………只是哭过以后，站起来，再勇敢地看一眼不会流泪的星空……” ; ; ; …………七年后，男孩和女孩的故事，在夜的星空下继续传说……
