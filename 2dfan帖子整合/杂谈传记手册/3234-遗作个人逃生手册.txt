
                遗作个人逃生手册(简易版) 　　这个游戏的目的是要使所有人从旧教室逃出，游戏中人物的关系错综复杂，任何一个看似不起眼的对话或动作都会对剧情产生变化，下面的提示只是提供逃出大楼的办法，至于和人物的互动关系就要靠自己了(实在是太多，光开头能选择的人就有六个)。 　　以下过程皆有顺序的，在找到一个重要线索前，下一件事是不会发生的.(原本随地可得的物品除外) 　　5F:Key Points to get KEY 　　找到机械油 → 在理科实验室见到 note &; 茶色的书 → 男厕见到保险箱 → 　　明美在讲桌找到 note → 拿茶色的书 → 开保险箱( 拿到 KEY ) → 开门. 　　*高岛老师上厕所时若不留下来等则高岛老师会被抓. 　　4F:Key Points to get KEY 　　看到黑板上的讯息 → 储藏室(拿起子) → 美术室(调查人像的头 &; 用起子敲头) → 5F(楼梯间见到 message) → 5F男厕开保险箱(用起子旋开螺丝可拿到电池) → 5F女厕(用电池 then take 　子) → 4F美术室(用 Hammer 敲头 to get KEY) → 开门 　　*记得拿红色油漆 　　*这一层要注意 里香，她会到处乱跑，小心不要让她被抓走. 　　3F:Key Points to get KEY 　　图书室(移动柜子，find Message &; 猫铃) → 4F视听室(猫铃给 琴末) → 储藏室(调查图案，use 起子 on wall，get U 形锁 key ) → 3F放送室隔壁(开 U 形锁) → 放送室(拿梯子) → 走道尽头(use 梯子，油，起子 开天花板的门，GET KEY) → 开门. 　　*这一层要注意宗光. 　　2F:Key Points to get KEY 　　厨房(看炉子，见KEY，形状记忆合金) → 男厕(酒精) → 3F(to open use 3F &; 4F KEY) → 见到 遗作 → 2F集合的教室(美由纪会跟来) → (到处走走) → 回教室(琴末不见) → 3F女厕(琴末) → (回去后) → 3F第一间教室(黑板数字 Message) → 放送室(对照表，时计　　，查时钟 find 打火机) → 2F厨 → 回教室(琴末不见) → 3F女厕(琴末) → (回去后) → 3F第一间教室(黑板数字 Message) → 放送室(对照表，时计　　，查时钟 find 打火机) → 2F厨房(use 书，酒精，打火机 in 炉子，GET KEY) → 开门 to 1F 　　*如果宗光有被抓，数字会变为两行，另一行为 机　中 　　Final Escpae from Building 　　见保健室汽油 → (到处走走，和 美由纪 talk) → 3F碰到 阵八(不让 美由纪跟 阵八，get KEY of 小门) → 放送室(叫 阵八待里面，U 形锁锁门) → 开小门上楼 → 和遗作说话(看 美由纪) → use 油漆 on 楼梯 → 回 3F 放 阵八→ 1F 拿遗作的刀 → 保健室(查最里面的窗子，用刀) → END 　　PS. 　　*不论游戏支线怎样进行，逃出来的方法都是一样的. 　　*最后一定可以救出所有人，不过结局就和先前的过程有关了.(如果过程中捡到不少录影带，那大概就不会有什麽好结局) 　　*要玩到 Happy Ending 的话，从头到尾都要小心对话和动作. 　 回标题
