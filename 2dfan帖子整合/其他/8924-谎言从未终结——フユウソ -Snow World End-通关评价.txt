
                <p align="center"><strong>谎言从未终结</strong>

<p align="center"><strong> ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;  ;&mdash;&mdash;フユウソ -Snow World End-通关评价</strong>

<p align="center"> ;

第一次接触分割商法游戏是Innocentgrey的<;flowers>;系列，不过当时还未出完该系列，所以到现在还全部未通关。当然有可能希望尽量抽出时间把余作通完。所以Campus谎言系列算是第一次通关的分割商法作品。

虽然官方说明过该系列每作相互独立，这点与<;flowers>;系列有所不同，而且确也在游戏制作上尽可能使剧情独立，但为了更好的理解游戏人物及背景，个人建议有兴趣的玩家尽可能把春、夏、秋、冬篇都通关一遍。下面简单发表下冬篇个人点评，希望对有兴趣的玩家带来帮助。

 ;

<strong>游戏评价</strong>

插画：★★★★★

音乐：★★★★☆

剧情：★★★★★

 ;

<strong>插画</strong>

插画方面不得不说比之前有了长足进步。同样是告白场景，春篇不仅笔画偏简陋，此外上色有些许油腻厚重，给人一种塑料光泽感；冬篇笔画变得精细，上色也更加清淡，人与自然融为一体。至于秋篇，エリス在游戏中定位应该属于男っぽい娘，或者说是大姐头， 但在CG上却变得萝莉化，给人一种违和感。另外，秋篇エリス反复出现食指指人的动作。至少在东方文化上，用手指人是非常不雅且不礼貌的，为何在游戏却频频出现，不得不说是槽点和减分项。

<strong>音乐</strong>

游戏中并没有提供音乐鉴赏功能，因此无法一一细细口味。但在通关游戏过程中，个人并没感到音乐方面的不协调感，也就是说整体差强人意，为了谨慎在这里打4分。

 ;

<strong>剧情赏析</strong>

&mdash;&mdash;是人都会说谎。

为了保护自身而说谎。为了否定自已而说谎。

为了欺骗自我而说谎。为了维持自尊而说谎。

谎言有很多种，谁都能找到说谎的理由。

 ;

而拥有仅听声音，就能看穿谎言的不成熟能力的少年，正是本作的主人公&mdash;&mdash;樱井宗一郎。

由于某次契机，主人公从魔法使那里获得了魔导具レッドライン，从而获得了看穿谎言的能力。但使用该能力是有代价的，因此他不得不成立“;猫之爪社团”;，并以此为旗号接受多项委托。而故事舞台正是围着接受委托这条线展开的。

这里需要强调一点是，尽管谎言系列背景具有共通性，但每作都是相互独立的，因此不应该将前作剧情<strong>全部</strong>带入后作，这里官方也给出了作品分歧点。

<img alt="" src="https://img.2dfan.com/uploads/intro/57107/d269b4c8b9a460fab988efa4a47e5fcd.jpg-watermark" style="width: 800px; height: 692px;" />

<strong>图六 谎言系列作品分歧点</strong>

也就是说，冬篇的故事背景应该是这样的：

春天，宗一郎在一次偶然中帮助了姬野樱月，导致她萌生爱意，为了确认樱井宗一郎的心意，于是前往“;猫之爪社团”;签订契约。但在该过程中契约被风吹走，因此没有成功签订。但也以此为契机，樱月加入了社团。

夏天，宗一郎及时反应，成功阻止了魔导具产生的蝙蝠进入和泉葵体内。葵一年前前往“;猫之爪社团”;监视该社团活动，在会长帝堂雪华撮合下，宗一郎与葵成为情侣。后因某件事导致分手，从此葵一直对他嗤之以鼻，宗一郎也自觉对不起葵，于是希望在她有困难的时候帮助她。也正是以此为契机，两人关系有所缓和。

秋天，从著名的吸血鬼家族カートレット离家出走的エリス，因为未能及时补充血液面临生命危险。宗一郎和エリス在四年前认识，由于宗一郎知晓了エリス的秘密，被迫同居。不久后エリス第一次血液危机，宗一郎通过签订契约，在スポーク帮助下成功找到备注。而四年后的今天，宗一郎和サージェント一起努力，成功找到血液，从而挽救了エリス。

冬天，由于姬野樱月受谣言中伤，宗一郎和エリス帮助樱月过程中，エリス因为教训了该同学而面临退学处分。走投无路情况下，宗一郎求助会长帝堂雪华，但条件是加入学生会，最后在会长帮助下成功解决问题。

从这里开始就是正篇。宗一郎发觉自己对会长的心意，于是约她在校园的“;恋樱”;见面，并向会长表白。会长接受了宗一郎的心意，两人成为情侣。

 ;一次偶然中，宗一郎得知“;恋樱”;了秘密：使用魔法的人陷入恋爱中，一年四季都可以看到该樱花绽放。于是宗一郎再次给会长在该地点见面，发现恋樱都是凋零的。宗一郎终于发现自己受到了会长的利用。原来会长同为魔导具レッドライン持有者，为了得到宗一郎，她设计让エリス面临退学，然后逼宗一郎求助于她，从而会长使用了契约，而契约条件是在12月20日前可以自由操控宗一郎的心意。之后会长以守住エリス的秘密为条件，继续让宗一郎留在身边，宗一郎唯有服从。

<img alt="" src="https://img.2dfan.com/uploads/intro/57107/3162d012039afdced4df517bfa3be2da.jpg-watermark" style="width: 800px; height: 450px;" />

<strong>图七 受操控宗一郎看到的恋樱</strong>

会长曾经认为自己拥有幸福的生活，但在得到レッドライン之后，发现一直以来人们都在欺骗她：父母希望把她作为政治道具，朋友希望接近她得到利益。从此会长对世界失去希望，并计划通过魔导具スノー&middot;ワールド&middot;エンド终结谎言的世界。最后，借助圣诞聚会，会长成功发动了魔法，前往阻止会长的宗一郎面临着二选一选项：要么杀死会长，要长终结谎言的世界。宗一郎没有选择其中一个，而是作了第三的选择&mdash;&mdash;杀死自己。最后会长终于屈服，解除了魔法。

<img alt="" src="https://img.2dfan.com/uploads/intro/57107/7311e5dcddab1b76a839382c104a74b3.jpg-watermark" style="width: 800px; height: 450px;" />

<strong>图八 ;会长逼宗一郎作出选择</strong>

<img alt="" src="https://img.2dfan.com/uploads/intro/57107/406fcf4c66b3309ef456aa1f535307ff.jpg-watermark" style="width: 800px; height: 450px;" />

<strong>图九 ;宗一郎给出答案</strong>

人物塑造上，不得不说宗一郎不算特别成功。作为一个魔导具レッドライン的拥有者，在长期的耳濡目染中，势必会对语言会产生一种敏感性，可以瞬间领会他人意图。在剧本中，宗一郎呈现的却是一种矛盾：一方面能瞬间看穿他人谎言，一方面却对女孩的示好浑然不觉。此外，在春、夏、秋篇中，宗一郎以“;善良”;为借口，为女孩提供帮助，也就是发生关系，虽然明知女孩对他有好感，但在事后并没有负起相应责任。说是帮助，不如说是伤害，这是剧情的减分项之一。幸好，在冬篇避免了这种局面。

会长方面，作为一个高中生，其野心显得与年龄格格不入。但仔细思考下游戏背景，就可以发觉这是能够理解的。会长出身名门，但在家族斗争中处于下风，于是父母希望把她作为政治道具重新掌权；而出身名门的她，见过了太多的谎言，从而对世界心灰意冷，这是使她发动魔法スノー&middot;ワールド&middot;エンド的原因。相比之下，宗一郎虽然自幼失母，但有着真正爱他的父亲，还有信赖他的朋友，因此尽管他不喜欢谎言，从不说谎言，但也不排斥谎言。正是他不说谎言这一点给会长带来的希望，并一步步喜欢上他，最后成功阻止了会长。
