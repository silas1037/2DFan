
                 ;  ;  ;  ;一开始看到trois公布新作，又炒冷饭，外加在DMM上看到只有900M的容量，瞬间兴趣全无，但是发售日还是抱着侥幸心理打开本作。其后在游戏过程中发现剧情非常精彩，激起我多年喜爱魔法少女之魂，同人作品的容量，过时的画风，也没能影响本作的实用性，全通后十分空虚，久久未能入睡，不禁感叹起我已经有多久没体验过类似的变身女主角剧情，在如今愈发纯爱的环境下，这种剧情大幅增强实用性的魔法少女作真的不知道什么时候才能再次体验到。<span style="color:#FF0000;">以下全是剧透。</span><span style="color:#FF0000;">由于我不懂日语，剧情有脑补成分，但八九不离十，毕竟是拔作。</span>

 ;  ;  ;  ; 主要人物介绍：前作女主有纱、前作幸存干部、本作新角色肥宅（变身女主偷拍狂）、新角色异世界黑化有纱、新角色异世界女主QB（一匹马）。肥宅和干部是合作关系，干部从肥宅身上吸取欲望能量，肥宅靠干部凌辱有纱从而偷拍自X。

 ;  ;  ;  ; 剧情梗概：剧情一开始，有纱战斗过程中QB突然消失，导致有纱战斗力大幅下降，完败，其后一段时间连哥布林等最弱魔物都打不过。视角切换到反派这边，干部能召唤魔物，对此肥宅提议干部能否尝试召唤一个变身状态的女主，干部尝试后却召唤出了QB，QB实力有点强，把反派二人吓退，有纱察觉到召唤兽反应后赶到现场和QB重逢，但是发现QB并不是她之前那只，而是异世界有纱的QB，不过有纱和异世界QB配合后也很强，成功击退了干部数次。干部打不过有纱，再次尝试召唤女主，这次成功召唤到了异世界的有纱，但是发现这个有纱就是异世界的QB原来的搭档，由于异世界有纱失去了QB，无限战败，黑化了（也更强了），于是就答应了加入反派方（加入的同时也戴上了反派给的戒指）。有纱在面对干部和黑有纱一起进攻，无力抵挡然后被抓回大本营，肥宅对有纱实施了本作第一个经典H（之后详细说明）。之后黑有纱感觉失去了对手，没有了目标，干部就让黑有纱去破坏城市，接着就出现了本作第一个选项，答应或不答应（答应的话就会进入最实用的线路，不答应就GE线路）。选择答应后，黑有纱去破坏城市，看到一般民众和警察虽无力抵抗，但也在尽力的反抗她的行为，黑有纱迷茫了，被肥宅趁虚而入迷晕了（这里有点扯，剧情需要）。醒来后发现自己被解除变身状态（因为戒指），接下来就是被肥宅无尽的H直至BE。

 ;  ;  ;  ;分析下本作实用性方面：本作的H出现都有前面剧情的铺垫，并不是单纯的战败H。本作出现了几个经典H场景，肥宅在女主被捕后脱下女主的胖次然后自X后再让女主把胖次穿上，这个H在学园退魔中也出现过，当时就让我印象深刻。还有一个是黑有纱被肥宅抓住后解除变身，黑有纱心里活动“;在之前的世界中被侵犯了无数次，但都是变身后的状态，人类状态被侵犯还是第一次”;，这里真的非常非常实用，主要因为前面剧情铺垫把黑有纱渲染的很强，反派二人都无法对她动手（肥宅也是先把黑有纱变身解除才敢动手）。本作有32个H，就不再一一阐述。

 ;  ;  ;  ;总结：本作告诉了我一个道理，同人的容量，过时的画风，只靠剧情也能给如今的我带来许久没体验过的心动，很久没玩过好的变身女主角凌辱作了。现在大环境都是流行纯爱，VB系列和催眠术系列这种都能越来越纯爱，而专注变身女主角的三角社新作也越来越水，剧情则是万年的反派装好人混入女主群（目测六月新作还是这样），真的是食之无味，弃之可惜。回想起上次这样的心动，记忆都有些模糊了，也许在推暗染复仇和超昂神骑也悄悄觉醒了一下，但是都还是达不到昨晚那种心动，真的是差点睡不着，激动了许久，今天没忍住在推特上用中文转发了assault老师的置顶，得到了中文的谢谢回复，可以说是gal生涯无憾了。
