
                 ;  ; 姑且说说章节的大致，因为故事是直接通到结局的，个人章节或许有两个结局，但不影响（死亡结局除外）

 ;  ; 前情提要：未来的49天，7个人平均分配了出来，男主角修一将与她们一起度过。

<span style="color:#ff00ff;"> ;  ; 忽那樱</span>便是推开第一章的主线人物，第一章中出现，她作为先驱者，来给大家的交流打开的一扇窗户，将傲娇蛮横的言叶，傻里傻气的茉莉，活泼强横的都瑠，沉默寡言的雫，团结到了一起，占用自己的时间，为她们提供在世间生活所必要的情报。她在起初的数天时间充当的是一个为他人着想的角色，就算时间如此之短，仍然使用自己的时间为大家铺好前进的路，不管是生前还是现在都在以姐姐身份在忍耐着各种痛苦，温柔、坚强、忍耐（吃货）便是她的标签。这个角色给我带来的感觉是敬佩，在这样的情况不仅为着大家着想还有着责任的意识，但就是这样的樱她也只不过是个不满10岁的孩子。普通农家里长大的她，离世时和其他人一样，她也有她自己的任性。她想听听家人的声音，她想在走向“;殿堂”;之前尝一尝家乡的萝卜，但这样会将其他六人暴露在危险中。即使如此，修一也是不断地在想办法满足樱的唯一一次的任性，在最后一天送樱安心的离开了这个才刚成立不久的大家庭。就算人已离去，她也在临走前教会了男主角修一享受生活又一种认识，为后面剧情推动带来更好的展开。

<span style="color:#FFD700;"> ;  ; 月野木言叶</span>是第二篇章跟随者先驱者樱的脚步，来到樱花镇体验生活的第二位主线人物。她和男主角的家庭背景相似，但家长选择却是截然相反的，尖锐的问题是两人交心且填补伤痛的契机，双方都羡慕着对方，但在互相体谅后却发现亲爱的人何尝不是备受痛苦与压力。她将自己以娇蛮任性伪装，让内心的痛不行于色。而内心却总以孤独为伴的她却在这片天地下，被修一看了个明白清楚。她的到来是男主角心中填补的良药，二人相互治愈着，也让相对于其他人不善于表达的修一有了更深的人生交流与认识。

<img alt="" src="https://img.2dfan.com/uploads/intro/9463/0fc5d8b7aed127d1b0f5c529ab408225.JPG?imageMogr2/quality/85!/thumbnail/1080x810>;" style="width: 470px; height: 235px;" />

 ;  ; 故事继续前进，来到<span style="color:#00FF00;">小鸟游茉莉</span>篇章。而初遇修一的她有着不同于她人的开朗与活泼，她在修一家里蹦蹦跳跳好奇地发问着。但修一却在这样的茉莉身上发现了她一直藏匿着苦楚。看似对什么都好奇的茉莉，其内心是一个对生活非常缺乏目标，而且没有正常童年的女孩。小鸟游茉莉的心理创伤问题，主要来源生前母亲对她的虐待。这份虐待以致她对“;母亲”;的认知产生了常人难以理解的偏差。她觉得大家的母亲都像自己家庭中那位一般可怕。所以她曾在属于自己的七天中的就主动提出过，想要逃离还未来到的压力，而提前结束自己的生活，但在后续大家的努力下茉莉第一次感受到活着的希望和期待，在结尾中和男主角的母亲上演一段感人催泪的离别剧。

<img alt="" src="https://img.2dfan.com/uploads/intro/9463/1ad5ad937523e86a4d72f9bfa59cfddf.JPG?imageMogr2/quality/85!/thumbnail/1080x810>;" style="width: 470px; height: 264px;" />

 ;  ; 故事第四章的这个七日是留给<span style="color:#FF0000;">百目鬼都瑠</span>的。性格粗放的她，想必在第一章就已经给玩家留下了一个比较有特点的印象。她有着与之豪放形象相符的天不怕地不怕的性格。虽然修一心理也多少有些迎接她到来的准备，但她最初的形象还是有点让修一始料未及的。打架、抽烟、飙车…;…;她所追求的东西，全是一些不良青年身上才有的不良习气。面对她的请求，一开始的修一也是有些不情愿的半推半就的答应了下来。但是，修一却通过与她在生活中接触后，发现了她其实是一名心中非常挂念朋友的女孩。粗犷却不蛮横，散漫却不失钻劲，强大却不凌弱，甚至是连身为男孩的修一都不曾能担当起的责任，她可以以简单的一句“;因为是朋友”;而全盘揽下。看着最后打完架后嘴里含着烟糖，悠然辞世的她修一又再一次陷入了沉思之中。

<img alt="" src="https://img.2dfan.com/uploads/intro/9463/f15094f5d67f67e24174145421c20e33.JPG?imageMogr2/quality/85!/thumbnail/1080x810>;" style="width: 470px; height: 263px;" />

 ;  ; 大家庭的人越来越少了，而这时的故事也终于来到了打的熊&mdash;&mdash;<span style="color:#00FFFF;">九十九雫</span>的篇章。一直以来都沉默寡言的雫，内心一直为了她的复仇计划而努力着。话不多说的她而雫其实是一名将整部故事矫向正轨，推入高潮的关键角色。性格较真的她，决定用自己的七天时间将和自己一行的六名少女的故事全部挖出。看着屡屡深夜还在在电脑前奋笔疾书的她，以及屡屡半夜因为噩梦而惊醒的她，修一虽知道无法替她背负些什么，但依旧是向她送去了自己的关怀方式。本是被仇恨所缠身的少女，而因为修一无微不至的关怀下看到了周围向她张开的怀抱。

<img alt="" src="https://img.2dfan.com/uploads/intro/9463/db8b9f9c8608bca714c7db1516dcc320.JPG?imageMogr2/quality/85!/thumbnail/1080x810>;" style="width: 470px; height: 262px;" />

 ;  ; 但故事依旧毫不停滞的来到了第六章。本以为故事会因为她的加入而变得吵吵闹闹，但是谁也不曾料到，一直以来那只色气满满的小公猫的本体却是一名假装成熟的普通少女。与其成熟打扮不同。她和其他女孩子一样，也渴望着自己能和恋人牵牵手，接接吻，一起相扶到老的恋情。以往的篇章里每名女主角都或多或少的存在其特有的孤僻或任性，而本章便是7days这部游戏中，最像极了一场真正恋爱的篇章&mdash;&mdash;<span style="color:#D3D3D3;">色摩宁音</span>篇。，作为最喜爱男主角的角色，虽然总说着黄段子想要做那种事，实际纯情的不行。看上去她作为最后一个走的人她就可以为所欲为，但这里是填完都瑠的坑，她也明白男主角的承受能力快要到达极限，所以自己作为最后一人来开解他，这也是她最大的心愿。这里面某种意义上男女感情的正牌理论描述，如释重负后却又有种窒息的感觉。

<img alt="" src="https://img.2dfan.com/uploads/intro/9463/dc42d341e916fe12cab90ca885d43f61.JPG?imageMogr2/quality/85!/thumbnail/1080x810>;" style="width: 470px; height: 265px;" />

 ;  ; 至于最后，虽然有点仓促但也算是大众最想要的结局了，就不剧透了。

 ;  ; 最后说下感想呗，虽然大致上都是压抑的感觉（可能是因为第一印象觉得结局会很悲伤吧），但在与角色互动的渲染下会笑出声来，当然这并不是什么多余的吐槽段子，而是充满着暖意的心之救赎，笑过之后迎来的便是“;鸡汤”;，对于男主角（我们）来说就是一次人生的思考。还有就是，里面的文本真的会很长，可与之相对的，质量也是同等，铺垫真是不少。虽然时间十分短暂，离别十分痛苦，但感受的那份悸动必然铭记于心，不管是里面的角色，还是融入剧情的同学。
