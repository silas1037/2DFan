
                要评价这游戏的话，这游戏简直是：社会恋爱剧情的“;硬核GAL”;！

社会上的恋爱并不像一般校园恋爱那么直接、单纯，校园恋爱里有很多都是单纯青春期荷尔蒙效应或者憧憬效应，不会深层次地考虑自己有没有真正意义上“;喜欢”;上对方，那种喜欢的感情就像空中楼阁，是没有任何基础与基石的，甚至很小的琐碎事情和进路方向问题都能引发大的争吵，所以大部分校园恋爱实际上很多都是以分手收场的。 ;
这游戏以一个比较高的次元角度，非常写实地描述社会上恋爱该有的模样。一般的校园恋爱剧情并不需要男主做什么有主见的事情，只要跟着剧情所需打打酱油就能走到结局。但是这套方法在社会上可就行不通了，社会上无论找工作、职场政治、居住、金钱管理、恋爱、花销，每一样都需要很强的个人主见、每件事都主动出击，因为再也没有手把手带你的人了，像学生时代那样混日子是行不通的，出来工作的各位应该都能明白我在说的意思。

Making＊Lovers这游戏剧情有一个很独特的特点：没有主线剧情。基本上开篇的选项点完就马上就能进入个人线剧情，个人线分为2大部分：进展篇与恋爱篇。这游戏我给予这么高的评价的原因，是因为剧本家对剧情的驾驭能力非常强大，虽然大部分是一些琐碎的恋爱剧情，但是这2位剧本家展现出了老练沉稳，称之匠心不为过的恋爱剧情驾驭能力。 ;
这游戏的剧情进展完全不缓不急、点到即止，全程都保证了绝对没有出现任何“;拖后腿、拖时间”;的无聊剧情，剧情全程都保持了很高的有趣和新鲜程度，逻辑清晰也不会有丝毫的衔接突兀，完全是一气呵成的剧情。并且这游戏每个可攻略角色的设定、兴趣爱好、生活模式、剧情进展都非常认真考究。虽然可攻略角色有5个这么多，但是却能让玩家享受到每个角色旗鼓相当水平的可爱与魅力，每个角色线路里完全没有一丁点凑字数剧情，打完每条线都能感受剧本家是多么认真写下的剧情。打完后你能感受到同一时期的其他的恋爱游戏剧情是多么的“;凑合”;和“;骗字数”;。

这游戏不仅每个可攻略角色的设定、兴趣爱好、生活模式、剧情进展都非常认真考究。更重要的地方是完全不会像很多公司纯粹只会在设定上下功夫，完全不会考虑到各项设定是否合理和是否与角色搭配，更不会考虑设定是否与剧情搭配，基本上就是什么“;什么设定看起来很厉害，还能勾起玩家好奇心”;就设定成什么。这样会导致角色行为模式非常死板，玩家也只是把角色当做是【被剧情设定的玩偶】来对待，你只有与“;芭比娃娃”;玩游戏的生硬感，而不是感受到一个“;活生生的女孩子”;的感觉。 ;
而Making＊Lovers在这方面非常严谨和考究，完全不会死靠“;设定”;来支撑剧情，剧本家会挖掘每个角色很多各种不起眼的行为模式与兴趣爱好的魅力点，你会在与女主角的交往过程中，一点一滴地发现女主角既普通、又不为人知的可爱一面，女主角也会在交往过程中发现“;你”;不为人知的另一面。这样的剧情简直就是现实中的恋爱的复刻版，每个角色都栩栩如生，你能感受到面前一个“;活生生的女孩子”;的真实感觉。

 ;

亜子线 ;
亜子线玩起来实在是太有趣又羞耻了，各种非常有趣的搞笑吐槽小剧场和互动，能保证任何人玩这条线都笑到合不拢嘴。 ;
【段子1】亚子：“;妈，我想养狗”; 老妈：“;狗的话那里不是有一只吗”; 老爹：“;汪汪汪！”; 亚子：“;这么恶心的狗才不会要啊！”; 【段子2】母亲:“;去摸那边那家伙的JJ。”; 亚子：“;好！”; 俺：“;？......啊！住手！”;（请让我再去笑一会） ;
交往后在有人的家中无耻地玩起了各种各样的羞耻play简直让人无法直视（笑。 ;
亜子这条线比较多地揭露了男主角的过去经历，包括为什么要离开家的原因，在挖出埋在家里后院的时光盒子后，忘记明白男主角当初就是因为暗恋亜子，为了杜绝忍不住对亜子出手的意外情况，在自愿在外独自租房子住做单身汉。看到这枚会对男主角为什么现在和亜子交往的过程顿生好感和祝福。 ;
而这条线的男主行为模式、心理年龄也由其他线的硬核职场剧情，变成一般常见的无需顾虑太多后果的校园恋爱游戏方向倾斜。与什么人交往就展现出相对应的行为，这与其他线强烈的反差不仅有趣而且方便剧情的进展，也能看出作者能文能武的强大剧本驾驭能力。

 ;

咲线 ;
咲线为我们讲述了男主因为联谊阴差阳错而结识了上流阶层家庭的大姐姐，并且男主角主动进入大姐姐的公司与大姐姐一起一边工作、一边交往、一边生活。 ;
咲这个角色描写得非常有魅力，内敛、自信、包容、有成年人该有的稳定个人价值观。 ;
老爸“;你丫的到底刷了多少个初始号才交到个这么好的女票(SSR)！”; 亜子：“;那我呢？”; 老爸“;你不就是杂鱼吗，当合成素材的那种！”; 这段话我能笑一年hhh~。 ;
但是个人觉得这条线有个明显的缺陷，就是男主角和咲的生活阶层差距比较远，无论同处职场上的很多剧情、同处生活里的很多剧情都有一定的疏离感，没有做到真正地消灭掉阶层不同带来的剧情隔阂。

 ;

玲菜线 ;
玲菜线展现了惊人的剧情自由度（笑，为了替代老妈的兼职顺便赚钱居然当了女生宿舍的管理员（笑死。在女生宿舍见到女孩子各种日常见不到的一面和各种趣事为剧情增添了不少色彩。 ;
玲菜的强势、有主见、机灵也成为了和男主交往的契机，因为男主作为普通人和玲菜工作的领域差别很远，所以剧情上也没有像其他4位女主角线一样安排男主与女主角在工作上有职场上的联系，也不会像其他4位女主一样天天黏到一起，约会与见面只安排男主与玲菜在职场以外的场景。由此可见剧本家的细致严谨，完全不会勉强去写成一样的套路，而是大胆写另一套完全不同的交往套路。 ;
但是我觉得这条线比起其他线来说剧情有点随便，也可以说有点都合主义，因为2人一开始生活的阶层差别太过遥远，工作和生活的圈子也差别太远，老实说能写成这样的剧情我觉得已经算是写得很好了。

 ;

真白线 ;
真白线前期的交往理由稍微有点莫名其妙（都合），但是除了交往理由以外，其他各个方面总体都写得比较好。2人作为邻居的各种趣事和生活百态，交往后一起在餐厅工作，一起约会、一起吃饭、一起玩游戏、经常到对方家里做客、叫对方起床，慢慢熟悉对方并且增进感情。 ;
虽然我说起来好像剧情显得老套，但是全靠剧本家强大的剧本驾驭能力的关系，全程剧情都保证了非常好的新鲜感和有趣感，这种不缓不急、循序渐进又点到即止的平淡恋爱剧情玩起来非常舒服有趣，简直来多少都嫌不够。 ;
真白稍微有点脱线的行为模式和声优的强大功力也让这角色的可爱程度倍增。

 ;

可怜线 ;
可怜线的剧情属于本游戏真真正正的精髓了，可怜这段交往写得太写实太有趣了，好几年都没见过写得这么好的恋爱桥段了。这条线实在太太太真实了，简直就是把现实中的场景完完全全还原到游戏里的感觉。为钱所困的2人被迫一起同居、被迫一起找新房子、“;半被迫”;交往、为了省被褥的钱准备睡一张床，同居后才慢慢发现互相喜欢。简直是我玩过所有恋爱游戏里最王道的剧情之一了。 ;
角色塑造非常巧妙，各种反差萌让角色显得极度可爱和富有魅力，可怜各种可爱的想法和对话能把玩家萌到全程“;流鼻血”;的状态。例如联谊结束跟男主分别那个场景：“;今天介绍的女孩是我真是抱歉了呢！”;（玩家血条已空！） ;
可以说这种恋爱才能说是真正意义上的“;恋爱”;，以前玩过的一切校园恋爱GALGAME在这条线面前都显得平面而单薄，如同纸做的玻璃一样经不起考验。可怜线中为我们描绘了一个非常甜美又有趣的社会中的爱情故事，这条线中的2人是真正从各种物质基础、工作、生活等多方面为前提，有好好地经营和培养爱情。男主也非常克制不会随便精虫上脑，每次可怜引诱男主的时候都能发现并且解决掉可怜的心结。男主和可怜也有深层次、多方面考虑了自己是不是真正喜欢对方。 ;
北见六花的配音实力在这条线也堪称逆天，把可怜这个略带傲娇、有点暴力、容易激动、又喜欢撒娇的角色魅力和丰满程度发挥到逆天程度，隔着屏幕你已经完全感受不到这是个GALGAME的“;角色”;，简直就是个活人在你面前的感觉。 ;
男主的人生观、价值观也是很符合一个社会人该有的样子，男主：“;我多少也有点存款，你一文不值的自尊心反而会让我为难。既然住在一起，就起码保证你有一个女性该有的生活的样子”;“;反正都要同住一段时间，我喜欢你能住得舒心些，在外忙碌一天回到的家居然是个糟心的地方，这不是最糟糕的吗”; ;
游戏中不像其他游戏一样对钱的作用、托关系找好工作这些世俗的东西做任何敷衍，而是全部有好好交代清楚，这些都是这游戏剧情含金量很高的地方。 ;
虽然这条线里男主失业，可怜也不是什么多高大上的白领职业，大家钱也不太多，穷穷地过日子。但是在【婚姻是2人互相扶持，而不是一个人在付出】这一理念上，可怜线写得非常好。大家互相包容对方的缺点，互相理解互相支持，互相尊重对方的职业（包括失业），不计较利益和得失，2人一起共同努力工作，钱多钱少无所谓，就近做点普通的工作，花更多时间陪在你身边已经很幸福。 ;
可以说这种纯洁而高尚的恋爱，无论主观上还是客观上都非常令人向往、憧憬和感人。

 ;

Making＊Lovers我认为已经可以称为新时代的“;秋之回忆”;这样的里程碑式恋爱游戏了。游戏保证做到了一个恋爱游戏该有的酸甜与有趣、与剧情的各方面平衡，游戏既有很多硬核的地方，绝不是一路硬核到底的剧情。 ;


这游戏不是强在用繁多的设定、庞大的世界观写出优秀的剧情；而是用普通又简单的设定写出同样优秀的剧情。前者好比用各种最贵最高档最新鲜的山珍海味做出的高档料理，后者则是把最简单的“;煎鸡蛋”;做成了高档料理，而后者明显难度更高。 ;


游戏里不仅每个角色的可爱与魅力写得非常优秀，让玩家对每个角色都喜爱到难以选择、难以割舍，而且每条线的剧情完成度都极高，故事也既有趣和创新，交往过程能体验到每位女主角都有毫不起眼、又和其他游戏完全不同的可爱魅力点。

 ;

对于凡是汉化过的纯爱类GALGAME基本全攻略过的我觉得：“;如果这游戏都不能称为恋爱游戏的里程碑式作品，那世上已经没有游戏能叫作恋爱游戏了！”;这游戏就是有这么强大的魅力，希望我的评论能为你的攻略带来帮助！
