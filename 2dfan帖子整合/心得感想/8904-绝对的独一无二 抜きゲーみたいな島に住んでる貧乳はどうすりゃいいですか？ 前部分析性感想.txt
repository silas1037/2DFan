
                并没有玩完，但是，比这部作品更令人印象深刻的，真的很少

几乎所有路人配音都想放第二遍，可能相同的作品找不到第二部了，我只能说文本构思真的很精彩，而玩过本作之后，可能会被装上奇怪词语组合的条件反射w，无法想象非日语版本，可能本作是那种不支持汉化的那一类。

 ; ;由于以下包括对一些情感动力的分析，可能比剧透更影响游戏体验，请谨慎阅读。（此类分析非常个人，不存在对错之说）

 ; ;先说人设吧，纯情的童贞男主来到了乱交碧蓝岛，上面全是碧池。环境描写的ero度已经达到了令人发指的程度，日常对性的想象张力简直是 ;脑洞大开。那么男主是干嘛的呢？

 ; 男主就负责纯情-童贞+吐槽，剧情设定存在着数不清的漏铜-比如怀孕了检不检测DNA，难道是国家拨款支持女性单亲抚养？原本来说，为什么会有这么多碧池，很多很多，而这一切应该由玩家在意吗？不，男主已经在吐槽了，男主就是这一设定的对立面，当玩家深想细节的时候，立场已经由某某脚本怎么这么蠢变成了男主接下来会做什么&mdash;&mdash;玩家的正反 心理动力被完美安置在将要冲突的铁轨上。

 ; ;说到人设，不得不先谈到妹妹，失去双亲后和妹妹相依为命 来到岛上，加上共同的“;纯情（普通百合）观念”;，看上去是可推很爽的角色，但实际上 ;妹妹是作为 ;对主角各方面描写的完善 ;而存在的（以写剧本视角）。（看到睡衣妹妹有点意动，之后迅速的 果体妹妹出现地那么的唐突其实就已经预见此角色的不可攻略性。）

 ; ;妹妹的作用是什么呢：其一，不用描写主角过多，反而描写与主角非常关系亲密的妹妹，不管是数学白痴+电脑天才的人设，还是对双亲的留念，以妹妹来展现要比主角一个人独白+全能+装傻 ;要来得好得多；其二是弥补主角与环境的巨大反差，由于主角的童贞吐槽役与环境是完全相反的，只有主角一个人的话冲突就太明显了&mdash;&mdash;不是觉得主角太做作就是觉得剧本太无脑，而每次都能让人听上两遍的超快语速妹妹吐槽 ;更贴近碧池环境的视角，也能让剧情的吐槽情绪更加平和娱乐（ばか兄妹だから）

 ; ;玩到这里，大部分人已经感觉不对了吧~ ;。对环境的ero描写从来没有以主角的第一视角心描出现过，而只是跟妹妹坚守着那一份‘;纯情’;。

 ; ;几乎所有的女角色都登场了一遍，而那些不是女主的女角色全部都被淹没在乱交碧池岛的ero背景中，萝莉女一跟沉默女三只是一笔带过，这时候第一个出现的人是谁呢？辣妹女二！在第二次登场解救男主的童贞危机后其实就已经大概猜得到了，这是第一个从乱交岛的氛围中脱颖而出 而与男主的纯情氛围相近的存在，虽然男主一直没有察觉到而吐槽她 反向积累情绪。

 ; 由于单方被认为是碧池，造成了女二客观上站在了一个比男主还纯情的位置上，也终于 把玩家进一步从 纯情负担中 慢慢解脱出来，世界的氛围好像没那么压抑了。就在这时候，女二面临着被识破+被强上的危机，全篇迎来了第一个高潮：那就是救黄毛逃亡，地下基地，正式宣战 中二爆表开始！

 ; OP2&mdash;&mdash;我们居然进入了萝莉女一视角~不正是之前路过男主家门口的美少女吗？故事的核心好像又为之一变：其实跟男主的情形处理是一样的，这个世界在负责ero，而身为幼女长相平板车的女一...（咳咳

 ; 我想玩到这玩家已经明白了，这个gal从头到尾都在做反差，把ero要素在环境中过度堆叠从而使“;纯爱”;的心情显得是那么的突出，女一作为一个幼女，为什么没人玩幼女play呢？班上的所有人都在热情似火地‘;干’;啊，为什么女一被忽视了？（之前提到的年龄限制？在这个乱交岛上小朋友早就被带坏了吧喂！）不，不，请看标题&mdash;&mdash;这时候我突然有种对不起女一的感觉，因为我虽然不讨厌幼女，但根本上来说是个巨乳党.

 ; 跟男主的童贞-自尊性 拒绝滥交不同，女一对性是暗暗憧憬的 ; 跟女二的装作碧池不同，女一俨然代表了另一种纯情，那是单纯的对性的一丝丝向往。看少女漫就很符合，但是折叠椅是什么意思？防暴？就在我这么想的时候，女一也遇到了与女二类似的场景，玩到这里我终于感叹：终于有以前玩galgame的感觉了。

 ; ;但是迅速画风一变，男主登场，战斗燃向&mdash;&mdash;让我想起了那一开始玩觉得搞笑的开头，现在，居然变得如此地能代表纯真，这种充满爽快的，不可言喻的 与ero战斗的快感

 ; ; ;居然，居然，在这个世界上居然存在！

 ; ;虽然目前为止每次女主都是被强被男主救，但是这作是神作我已确定无疑（即使第一个Hシーン还没玩到）

 ; ;很期待剩下的剧情展开，因为如此美丽宽阔的前路铺好了之后，很难想象能坏到哪里去。
